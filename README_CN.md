# BaseUtil
 
# 本框架主要用于Android快速开发，集成大部分常用API，让大多数人能够集成此Model就能马上入手开发。
# 使用方法：
在Application中注册：MainInit.getInstance().init(this);
# 功能注册：
```
BaseConfig.setDeBug(true);
BaseConfig.setBackFinish(false);
....
```
